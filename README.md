# 五、功能详细说明

## 5.1 基础单图生成（gpt_image_api.py）

### 功能

| 功能 | 说明 |
|------|------|
| 自然语言生图 | 一句话描述，生成图片 |
| 比例快捷词 | `竖版`→3:4 / `横版`→16:9 / `正方`→1:1 / `9:16`→竖屏 |
| 中文场景名 | 自动转换（如"竖版"→`aspect_ratio: 3:4`）|
| 参考图支持 | 支持本地路径或URL |
| 质量保证词 | 自动附加"最佳质量"等增强词 |

### 使用方式

```bash
# 基础生成
python3 gpt_image_api.py "一只可爱的橘猫"

# 指定比例
python3 gpt_image_api.py "风景照" --ratio "16:9"

# 使用Shell脚本
./gen_image.sh "一只可爱的橘猫"
./gen_image.sh "风景照" --ratio 16:9
```

### 比例门禁规则

| 比例 | 是否需要确认 | 说明 |
|------|-------------|------|
| 1:1（正方形）| ❌ 不需要 | 默认比例，直接放行 |
| 3:4（竖版）| ✅ 需要 | 需显式确认 |
| 16:9（横版）| ✅ 需要 | 需显式确认 |
| 9:16（竖屏）| ✅ 需要 | 需显式确认 |

**门禁逻辑**：除1:1外，其他比例必须通过参数传入，否则自动拦截并提示。

## 5.2 参考图身份锚定（check_reference.py）

### 功能

| 功能 | 说明 |
|------|------|
| 参考图压缩 | >8MB → 自动压缩到768px，原图不动 |
| 格式验证 | 仅支持jpg/jpeg/png/webp |
| 尺寸检查 | 宽高均>100px |
| 身份锚定 | 传给GPT Image保持角色一致性 |

### 使用方式

```python
from check_reference import compress_reference_image

# 压缩参考图
result = compress_reference_image("input.jpg", "output.jpg")
# 返回: {"path": "...", "size": "...", "was_compressed": True/False}
```

## 5.3 常子涵肖像模板（portrait_prompt.py）

### 功能

| 功能 | 说明 |
|------|------|
| 三庭五眼约束 | 面部的上中下庭 + 五眼距离精确描述 |
| 侧脸禁止 | 90°纯侧脸自动拒绝 |
| 一致性保证 | identity_anchor保持跨次生成相似度 |
| 风格支持 | cinematic / natural / soft_light / dreamy |

### 肖像模板字段

```
身份锚定（identity_anchor）：常子涵的三庭五眼数字比例
面部结构：三庭（发际线到眉心/眉心到鼻尖/鼻尖到下巴）
眼睛：杏眼、大而圆、眼角尖锐
下巴：短下巴，圆弧
嘴巴：M唇，唇珠分明
鼻子：塌鼻子
```

### 使用方式

```python
from portrait_prompt import build_portrait_prompt

# 构建常子涵肖像提示词
prompt = build_portrait_prompt(
    description="穿着校服，在教室里",
    style="cinematic",
    reference_image="path/to/ref.jpg"
)
```

## 5.4 PLOC手账涂鸦Pipeline

### 功能

| 功能 | 说明 |
|------|------|
| VLM识别 | 识别画面元素（湖/山/云/建筑等）|
| 文案生成 | 根据识别结果生成3-5条繁体大字标注 |
| 涂鸦生成 | GPT Image生成白线手绘风格装饰 |
| 人工筛选 | 最终成片人工挑选最优 |

### Pipeline流程

```
Step 1 → VLM识别画面元素（ollama_vlm.py）
Step 2 → 根据识别结果撰写标注文案
Step 3 → PLOC手账涂鸦生成（gpt_image_api + ploc_style）
Step 4 → 人工筛选最优成片
```

### PLOC风格模板（固定不变）

```
ins風格治愈手賬裝飾，白色纖細手繪線條勾勒畫面裡所有物體輪廓，
搭配虛線、小圓圈可愛箭頭指向景物，字體要大、要粗、要清晰明顯，
起筆收筆痕跡，鋼筆書寫質感，不改變原圖畫面、不改動原圖色調，
只增加白色手繪標注裝飾，溫柔日系氛圍感。
```

### 标注规则

| 正确 ✅ | 错误 ❌ |
|---------|---------|
| 花園時光 ✦ 玫瑰與少年 ✦ 城市綠洲 | 黑Polo ✦ 白褲 ✦ 銀手表 |
| 场景意象词+情绪词 | 直接列穿搭/首饰/具体物品 |

### 比例

- 标准：3:4（1024×1365）
- 参考图：原图不变

### ⚠️ 重要教训

1. **大图必须先压缩再用VLM识别**（>1MB建议压缩到512px宽）
2. **必须用ollama_vlm.py标准化脚本**，禁止用模型直接读图
3. **PLOC模板已注册到Pipeline模板库**，通过Pipeline调用时自动执行门禁

# 附录B：常见问题

**Q1: 提示"未配置API Key"**
A: 检查.env文件是否存在且包含GRSAPI_KEY，或运行 `export GRSAPI_KEY=your-key-here`

**Q2: 参考图生成的角色不一致**
A: 确保参考图是正面或微侧角度（非90°侧脸），且文件格式为jpg/png/webp

**Q3: 生成失败，提示"侧脸禁止"**
A: 不能用纯侧脸图片作为参考图，请换用正面或微侧脸照片

**Q4: 图片比例不对**
A: 生图时必须传入 `--ratio` 参数（如 `--ratio 16:9` 或 `--ratio 3:4`）

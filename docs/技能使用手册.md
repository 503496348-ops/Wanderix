## 1. 快速开始

### 首次使用（3步）

```bash
# 1. 克隆/下载本仓库到本地
git clone <repo-url> 艺游未境-Wanderix/
cd 艺游未境-Wanderix/

# 2. 配置 API Key
cp .env.example .env
# 编辑 .env，填入你的 GRSAPI_KEY

# 3. 生成第一张图片
./gen_image.sh "一只橘猫在草地上晒太阳"
```

### 零配置测试（使用智障引导）

如果直接运行 `./gen_image.sh`，脚本会引导你完成配置：

```
$ ./gen_image.sh "一只橘猫"
❌ GRSAPI_KEY 未设置
→ 请访问 https://yh.grsai.ai 注册获取 Key
```

## 2. 功能详解

### 2.1 基础生图

```bash
# 默认正方形 1024×1024
./gen_image.sh "一只橘猫"

# 指定像素尺寸
./gen_image.sh "旅行照片" "1920x1080"

# 指定比例
./gen_image.sh "小红书配图" "3:4"
```

### 2.2 参考图生图

保持角色/物体一致性，携带参考图：

```bash
./gen_image.sh "财阀千金肖像" "2:3" ~/Desktop/ref2.jpg
```

Python API：

```python
from gpt_image_api import generate_image

result = generate_image(
    prompt="财阀千金肖像",
    size_raw="2:3",
    reference_images=["/path/to/ref2.jpg"]
)
```

### 2.3 输出文件名前缀

```bash
./gen_image.sh "新疆之旅" "3:4" -n "毕业季海报"
# 输出: 毕业季海报_20260516_ab12.png
```

| 快捷词 | 像素 | 比例 | 场景 |

## 4. 参考图体系

### 常子涵肖像参考图

| 编号 | 文件 | 用途 | 大小 | 首选 |
|------|------|------|------|------|
| ref2.jpg | ref2.jpg | 淡妆正脸自拍 | 117KB | ✅ |
| ref3_side.jpg | ref3_side.jpg | 侧脸参考（鼻子/下巴线条） | 67KB | 辅助 |
| ref1.png | ref1.png | 偏分刘海妆后自拍 | 1.9MB | 备选 |

**使用规则：**
- 单张生成 → 只用 ref2.jpg（最稳定）
- 双参考图 → ref2.jpg + ref3_side.jpg
- ❌ 禁止：无参考图生成常子涵肖像

### 大图自动压缩

> 参考图 >8MB 时，自动复制到 `/tmp` 压缩，上传后清理，原图不动。

## 5. Pipeline自动化

### PLOC_治愈手账涂鸦

为旅行/风景照片添加手账装饰风格。

**流程：**

```bash
# Step 1: VLM 识别画面元素
cd ~/.openclaw/skills
uv run python ollama_vlm.py "列出图片中可标注的元素" /path/to/photo.jpg

# Step 2: 运行 Pipeline
python pipeline/engine.py run PLOC_治愈手账涂鸦 \
  --input /path/to/photo.jpg \
  --output ~/Desktop/手账涂鸦.png
```

**标注文案规则：**
- ✅ 正确：`花園時光 ✦ 玫瑰與少年 ✦ 城市綠洲`
- ❌ 错误：`黑Polo ✦ 白褲 ✦ 銀手表`（直接列物品清单）

### 列出所有模板

```bash
python pipeline/engine.py list
```

## 6. 强制规定

| 编号 | 规定 | 说明 |
|------|------|------|
| F1 | 不可改动用户提示词 | 用户描述原封不动传给 API |
| F2 | 固定模板不可擅自修改 | 身份锚定、negative 约束等 |
| F3 | 常子涵肖像必须带参考图 | 禁止无依据单图生成 |
| F4 | 禁止 90° 纯侧脸 | 一致性严重下降 |
| F5 | 正方形需确认比例 | 比例门禁拦截 |

## 7. 故障排查

| 症状 | 原因 | 解决方法 |
|------|------|---------|
| `GRSAPI_KEY 未设置` | .env 未配置 | 复制 `.env.example` 为 `.env` 并填入 Key |
| `insufficient credits` | 积分不足 | 访问 https://yh.grsai.ai 充值 |
| `Permission denied` | 脚本无执行权限 | `chmod +x gen_image.sh` |
| 生成超时 | 网络慢/队列长 | 等待重试（最多5分钟） |
| 参考图质量差 | 人脸过小/模糊 | 使用 `check_reference.py` 诊断并裁切 |
